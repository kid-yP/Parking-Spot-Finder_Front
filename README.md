# Parking-Spot-Finder Frontend

The Parking-Spot-Finder Frontend is a mobile application built with Flutter that connects to the backend service to deliver a seamless parking management experience. It allows users to authenticate securely, view available parking spots, and make reservations with an intuitive and responsive UI.

## 🚀 Features
User Authentication: Login and signup with secure token-based access

Parking Spot Search: Real-time availability display with geolocation filtering

Reservation Management: Book, cancel, and view reservations easily

Cross-Platform: Runs smoothly on both Android and iOS

Integration Ready: Works with the Parking-Spot-Finder backend API

## 🛠️ Tech Stack
Flutter (frontend framework)

Dart (programming language)

Provider / Riverpod (state management, depending on your setup)

HTTP package (API communication)
